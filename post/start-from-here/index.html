Start From Here

Sep 25, 2014

行行重行行，朝朝复朝朝。

做技术也有好几年了，其间每每有”顿悟”的时候，就想把他记录下来：一是告知世人，我又顿悟了，我屌爆了； 二是我不能确定我的大脑是否能永远记住这些东西。

我总在计划，也总在变化。计划了很久的搭建个人博客，有好多的想法，也总在变化着设计的想法。做为一个 强迫症患者，一个选择困难症患者，一个天秤座的男纸，在计划没有完美之前，是不会下手去干的。但时间真的是 是一种神奇的东西，有人说他是疗伤的圣药，有人说他是一切的解药；但不管他是什么药，他改变了我的想法，这 样看来，他是治疗强迫症的药。我还没有痊愈，但我还是决定将这个没有完美的计划执行下来，搭建了这个博客。 以后会发生什么？我不知道。但不管怎样，总算开始了：

let’s start from here 无所谓慢慢来 迷宫一样的未来 ……
